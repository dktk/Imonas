Nginx SETUP
    - sudo vi /etc/nginx/sites-available/test.imonas.com

            server {
            listen 80;
            server_name test.imonas.com;

            location / {
                proxy_pass         http://127.0.0.1:5000;
                proxy_http_version 1.1;

                proxy_set_header   Host              $host;
                proxy_set_header   X-Real-IP         $remote_addr;
                proxy_set_header   X-Forwarded-For   $proxy_add_x_forwarded_for;
                proxy_set_header   X-Forwarded-Proto $scheme;

                proxy_set_header   Upgrade           $http_upgrade;
                proxy_set_header   Connection        $connection_upgrade;
            }
        }

    - sudo ln -s /etc/nginx/sites-available/test.imonas.com /etc/nginx/sites-enabled/test.imonas.com
    - Websocket helper
        Check if you already have it: grep -n "map \$http_upgrade \$connection_upgrade" /etc/nginx/nginx.conf
        Add content to: /etc/nginx/nginx.conf (if not there)
            map $http_upgrade $connection_upgrade {
                default upgrade;
                ''      close;
            }
